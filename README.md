AutoContent AI is a fully automated content creation platform that leverages AI (Google Gemini, OpenAI, or Custom LLMs) to generate, schedule, and publish content for YouTube, TikTok, and Facebook.

## Features

-   **Multi-Platform Support**: Generate content for YouTube, TikTok, and Facebook.
-   **AI-Powered**: Uses Google Gemini, OpenAI, or compatible local LLMs to create scripts, captions, and hashtags.
-   **Visual Branding**: Customize fonts, colors, and background styles.
-   **Automation Pipeline**: One-click generation of niche-specific content.
-   **Dashboard**: Track content performance and scheduled posts.

## Getting Started

### Prerequisites

-   Node.js 18+
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/autocontent-ai.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

### Configuration

1.  Go to `Settings` in the dashboard.
2.  Configure your AI Provider (Gemini, OpenAI, or Custom).
3.  Set up your social media API keys (mocked for now).
4.  Customize your Visual Branding.

## Deployment

To deploy this application, you can use Vercel:

1.  Push this repository to GitHub.
2.  Import the project into Vercel.
3.  Vercel will automatically detect Next.js and deploy.

## Tech Stack

-   **Framework**: Next.js 14
-   **Language**: TypeScript
-   **Styling**: CSS Modules / Global CSS
-   **AI**: Google Generative AI SDK, OpenAI API Support
